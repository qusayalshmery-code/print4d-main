# مطبعة البُعد الرابع - الموقع الجديد

## 📋 نظرة عامة

موقع متكامل لمطبعة البُعد الرابعة في النجف الأشرف، يحتوي على 6 أقسام رئيسية وإدارة العروض السريعة.

## 🏗️ هيكل المشروع

```
/
├── index.html              # الصفحة الرئيسية
├── data/
│   ├── config.json         # إعدادات الموقع الرئيسية
│   └── offers.json         # العروض الحالية
├── assets/
│   ├── css/main.css        # الستايل الرئيسي
│   ├── js/main.js          # السكريبت الرئيسي
│   ├── images/             # الصور
│   └── videos/             # الفيديوهات
├── snapshots/              # النسخ الاحتياطية
├── admin/                  # (لاحقاً) لوحة التحكم
├── products/               # صفحات المنتجات
├── library/                # مكتبة الكتب
└── offers/                 # صفحات العروض
```
